# March-Madness-Tournament-Winner-Predictor
My first crack at data analysis. This simple code just scrapes data and uses functions to filter and clean data into one CSV file. Future iterations will go more in depth with data normalization and predictive modeling. Code snippet that scrapes data from Sportsreference.com to gather data from the 2026 NCAA men's DI field. It filters and reads to a CSV file for analysis. Repository also contains excel file data from Kenpom.com and compiled historical data from previous winners.

# HOW TO RUN CODE
The code included is designed to scrape data from Sportsreference.com. However, some of the data used in this analysis also came from Kenpom.com, which I couldn't scrape since I don't have a subscription. In order for the code to run properly, save the "kenpom_stats.csv" file to the same folder as the code in order to run successfully. Next years iteration will run better (this is my first data analysis project and I was learning on the fly)
